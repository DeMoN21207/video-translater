# Video Translater Desktop

Приложение для локальной транскрибации видео и аудио при помощи моделей Whisper с поддержкой разбивки на диалоги.

## Возможности

- 📂 Загрузка видео- и аудиофайлов прямо из десктопного интерфейса.
- 🧠 Выбор размера модели Whisper (включая автоматический подбор) и повторное использование уже загруженных весов из кэша.
- ⚡ Настройка режима вычислений: авто, только CPU или только GPU.
- 🗣️ Формирование диалогов по сегментам, попытка группировки по голосам (при наличии `pyannote.audio`) или разбиение длинных реплик на цитаты.
- 📊 Наглядный прогресс-бар с отображением этапов обработки.
- 📝 Просмотр полной расшифровки и таблицы с временными метками и спикерами в одном окне.

## Установка

1. Установите PyTorch (GPU или CPU версия):

   ```bash
   # GPU версия (пример для CUDA 11.8)
   pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118

   # или CPU версия
   pip install torch torchvision torchaudio
   ```

2. Установите остальные зависимости:

   ```bash
   pip install -r requirements.txt
   ```

   > Для режима «По голосам» дополнительно требуется токен Hugging Face (`PYANNOTE_AUTH_TOKEN`).

## Запуск интерфейса

```bash
python app.py
```

Первый запуск выбранной модели займет больше времени — веса Whisper будут скачаны и закэшированы в `~/.cache/whisper`. Повторные запуски используют локальный кэш.

### Поля интерфейса

- **Файл** — выберите видео или аудио.
- **Тип входного файла** — сообщите приложению, что вы загрузили (нужно для пропуска лишних конвертаций).
- **Модель Whisper** — от `tiny` до `large-v3` или автоматический выбор.
- **Ускорение** — авто / только GPU / только CPU.
- **Формат диалога** — вывод сегментов, группировка по голосам (beta) или разбиение длинных реплик на цитаты.

Пока идет обработка, прогресс-бар показывает текущий этап. Если требуется диаризация, убедитесь, что установлена библиотека `pyannote.audio` и задан токен.

## Командная строка

CLI остался для сценариев без графического интерфейса:

```bash
python transcribe_video.py путь/к/файлу.mp4
```

Результат сохраняется в папке `translates` рядом со скриптом (или в директории, указанной в `config.json`).

## Конфигурация

Измените значения в `config.json`, чтобы настроить:

- `save_path` — каталог для сохранения результатов;
- `include_timestamps` — выводить временные метки;
- `use_cuda` — предпочтительный тип устройства (`auto`/`gpu`/`cpu`);
- `model_size` — модель Whisper по умолчанию;
- `delete_audio` — удалять ли временные аудиофайлы;
- `language` — язык распознавания;
- `input_type` — тип входного файла по умолчанию (`video` или `audio`);
- `dialogue_mode` — режим диалогов (`segments`, `speakers`, `quotes`);
- `pyannote_token` — токен для спикерной диаризации (можно оставить `null` и задавать через переменную окружения);
- `speaker_label_prefix` — префикс названия спикеров в выводе.

## Устранение неполадок

- **Нужна диаризация, но нет токена:** зарегистрируйтесь на Hugging Face, создайте токен и задайте переменную окружения `PYANNOTE_AUTH_TOKEN`.
- **Недостаточно VRAM:** выберите меньшую модель Whisper.
- **GPU не определяется:** убедитесь, что установлен PyTorch с поддержкой CUDA, и обновите драйверы видеокарты.

Приложение не требует Docker и работает полностью локально.
